# Персональный сайт Гульшат Ахмадиевой

Персональный сайт аналитика бизнес-процессов и специалиста по ИИ-автоматизации.

Проект представляет профессиональный опыт, прикладные решения, реализованные кейсы и авторские цифровые проекты.

## Опубликованная версия

**Основной сайт:**
https://gulshat-akhmadieva.github.io/gulshat-personal-site/

**Музыкальная комната:**
https://gulshat-akhmadieva.github.io/gulshat-personal-site/music.html

## Что представлено на сайте

* анализ и автоматизация бизнес-процессов;
* автоматизированная отчётность Wildberries и Ozon;
* решения для контроля качества и клиентских обращений;
* Telegram-консультанты для бизнеса;
* профессиональный опыт и подтверждающие материалы;
* реальные отзывы и фрагменты переписок;
* музыкальная комната с персональными песнями.

## Особенности проекта

* адаптивная версия для смартфонов, планшетов и компьютеров;
* мобильное меню;
* модальные окна и лайтбоксы;
* оптимизированные изображения WebP;
* аудиоплееры для авторских композиций;
* Open Graph-превью для публикации ссылок;
* семантическая HTML-разметка и базовая SEO-оптимизация.

## Технологии

* HTML5
* CSS3
* JavaScript
* GitHub Pages

Проект создан без фреймворков и систем сборки.

## Структура

* `index.html` — главная страница;
* `music.html` — музыкальная комната;
* `styles.css` и `music.css` — оформление страниц;
* `script.js` и `music.js` — интерактивные элементы;
* `assets/` — изображения, иконки и аудиофайлы.
